# 推理审计 — 三角一致性检查

## 铁律

```
证据、推理、决策三者必须一致，否则不开仓
```

**灵感来源:** Trade-R1 (三角一致性指标) + MemGuard-Alpha (记忆污染检测, 清洁信号收益是污染信号的 7 倍)

## 三角一致性检查

开仓前，回答以下 3 个问题。任何一个答案为"否" → 不开仓。

### 1. 证据 → 推理 一致吗？

```
我收集到的数据是否真的支持我的分析？
```

**检查项:**
- RSI 说"超卖"，但 EMA 趋势向下 → 超卖不等于反弹
- OI 上升 + 价格下跌，但 OI 变化只有 2% → 这个变化量是否异常？(查看 OI 百分位)
- "关键支撑位" 是布林下轨 → 布林带在趋势市下移，下轨不是真支撑
- 多空比 > 2.0 → 但大户多空比方向相反 → 散户指标可能误导

**常见失败模式:**
| 证据 | 错误推理 | 正确推理 |
|------|---------|---------|
| RSI < 30 | "肯定反弹" | "可能继续超卖到 15" |
| OI +3% | "空头拥挤" | "3% 可能是正常波动" |
| Funding 负 | "空头过度" | "可能趋势转空，funding 合理" |

### 2. 推理 → 决策 一致吗？

```
我的分析是否真的支持 "现在开仓" 这个决策？
```

**检查项:**
- 分析说"可能反弹" → "可能" 够不够开仓？需要 "大概率"
- 分析说"趋势向下但超卖" → 超卖的逆势交易需要更高的评分门槛
- 时间框架是否匹配？(1h 超卖但 4h 趋势向下 = 短期反弹 vs 中期下跌)
- 赔率 1.6:1 通过了 1.5 门槛，但这笔交易真的值得吗？

**关键问题:** "如果我不看这个币，随机给一个人看同样的数据，他会不会也开这个仓？"
- 如果答案是 "可能不会" → 你可能有偏见

### 3. 这个判断是当前数据驱动的，还是记忆模式？

```
我的预测是基于眼前的数据分析，还是在匹配记忆中的历史模式？
```

**LLM 特有风险 (MemGuard-Alpha 发现):**
- LLM 训练数据中包含大量历史行情
- "这个走势很像 2024 年 3 月的 SOL" → 可能是记忆匹配，不是分析
- 记忆匹配的信号收益只有清洁信号的 1/7

**检查项:**
- 我能不能只用当前数据（不引用任何历史事件）来论证这笔交易？
- 如果我去掉"类比历史"这个论点，剩下的论据还够不够？
- 这个币种是否处于训练数据截止后的新状态？(如果是，更可靠)

## 输出

```
🔍 推理审计 — {symbol} {方向}
━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣ 证据→推理: ✅ 一致 / ❌ 不一致
   {一句话说明}

2️⃣ 推理→决策: ✅ 一致 / ❌ 不一致
   {一句话说明}

3️⃣ 数据驱动 vs 记忆匹配: ✅ 数据驱动 / ⚠️ 可能含记忆偏差
   {一句话说明}

→ {通过 / 不通过 — 理由}
```

## 在什么情况下应该 FAIL

- 核心论据依赖布林带，但当前是趋势市
- OI 变化不到 5% 但给了 +2 分
- 理由中出现"上次也是这样"、"类似 XX 年"等历史类比
- 牛方论证中超过一半的论点是"感觉"而非数据
- 同一方向已有 2 个持仓且高度相关

## Red Flags

- "数据支持我的直觉" → 直觉不是数据
- "所有信号都对齐了" → 检查是否在确认偏误
- "赶紧过审计，行情要跑了" → 急躁是最大的 Red Flag
- 审计只花了 10 秒 → 没认真做

**以上全部意味着：停下来。重新审视。**
