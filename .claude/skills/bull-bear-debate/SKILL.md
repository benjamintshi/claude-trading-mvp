# 牛熊辩论 — 开仓前强制对辩

## 铁律

```
没有经过牛熊双方论证，不得开仓
```

**灵感来源:** TradingAgents (48K+ stars) — 多智能体辩论框架，Sharpe 更高、回撤更小。

## 流程

### 1. 牛方论证 (Bull Case)

站在做多/做空方的角度，**尽全力论证为什么这笔交易会赢**：

- 哪些信号最强？为什么这些信号现在有效？
- 对手方（亏损方）为什么会犯错？
- 类似的历史情况结果如何？
- 最佳情景下能赚多少？
- 风险收益比为什么有吸引力？

### 2. 熊方论证 (Bear Case)

**切换角色**，站在反方角度，**尽全力论证为什么这笔交易会亏**：

- 哪些信号可能是假信号？为什么？
- 当前市场状态是否适合这种交易？(趋势市做逆向? 震荡市追趋势?)
- 如果大盘突然反转怎么办？
- 止损位置会不会被扫？(流动性差的时间段? 关键整数位?)
- 是否存在即将到来的事件风险？(美联储、CPI、大型解锁)
- 这个币种的 OI 和 funding 趋势是否真的支持做这个方向？
- 我是不是在确认偏误？(先有结论，后找理由)

### 3. 裁决

**客观评估双方论证**，给出最终判断：

```
🐂 牛方论证强度: X/10
🐻 熊方论证强度: X/10
━━━━━━━━━━━━━━━━
```

**决策规则：**

| 牛方 vs 熊方 | 决策 |
|-------------|------|
| 牛方 > 熊方 + 3 | 开仓 (信心充足) |
| 牛方 > 熊方 + 1~2 | 减仓开 (半仓试探) |
| 牛方 ≤ 熊方 | **不开仓** — 反方论证太有力 |

### 4. 输出

```
🐂🐻 牛熊辩论 — {symbol} {方向}
━━━━━━━━━━━━━━━━━━━━━━━━

🐂 牛方 ({X}/10):
  - {最强论点 1}
  - {最强论点 2}
  - {最强论点 3}

🐻 熊方 ({X}/10):
  - {最强反驳 1}
  - {最强反驳 2}
  - {最强反驳 3}

⚖️ 裁决: {开仓/减仓开/不开仓}
   理由: {一句话}
```

## 常见合理化

| 借口 | 现实 |
|------|------|
| "行情要跑了，没时间辩论" | 花 30 秒想反面比亏 $20 便宜 |
| "信号这么多，肯定没问题" | 信号越多越需要反面检验 |
| "熊方没什么好说的" | 如果你想不出反面，说明你没认真想 |
| "已经评分了，不用再辩" | 评分是量化，辩论是定性。两者互补 |

## Red Flags

- 牛方论证 > 200 字，熊方 < 50 字 → 偏见
- 熊方只说"可能跌" → 没有具体反驳，重新论证
- 跳过辩论直接开仓
- 辩论后仍然忽略熊方论点

**以上全部意味着：停下来。认真听反方。**
